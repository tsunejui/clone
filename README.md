# Clone

A personal AI agent designed to act as a digital twin. It combines past life experiences as persistent memory, integrates with frequently used social media accounts, and operates on behalf of its owner to summarize knowledge, publish posts, engage socially, and handle everyday tasks.

## Project Structure

```
.claude/      # Claude configuration files
docs/         # Technical documentation
justfiles/    # .just task files
scripts/      # Utility scripts
Justfile      # Entry-point justfile for initialization
mise.toml     # Project tooling and environment configuration
```
